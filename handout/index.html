Skills

/aaa Evaluate and improve ideas, features, architecture, and code with evidence, gap analysis, and 3–4 world-class options. /documentation-standard Create, update, and refactor architecture docs, ADRs, user manuals, and development guides following enforced structure and naming conventions. /llm-wiki Set up or operate a general-domain LLM Wiki — a persistent, LLM-maintained markdown knowledge base for research, personal knowledge, or any non-product domain. /llm-wiki-product Set up or operate a product-team LLM Wiki for software-project content: features, specs, roadmaps, architectural decisions, and code notes. /plan-maker Creates and updates detailed, actionable implementation plans with the smallest possible tasks, full test specifications, and explicit dependency chains. /skill-packager Packages a Claude Code skill folder as an upload-ready ZIP for Claude Desktop or claude.ai, ensuring the ZIP's top-level folder matches the skill's name frontmatter.

Commands

/da-review Single-pass devil's advocate review that finds every flaw in a target — code, plan, diff, or design — without applying any fixes. /feature-refinement An interactive senior product thinker that guides you from a rough feature idea to a well-scoped Feature Brief ready for /plan-maker. /implement-all Repeatedly delegates to /implement-next — one task at a time — until every uncompleted task in a plan file is implemented, verified, and committed. /implement-next Reads a plan file, implements the next uncompleted task with strict TDD, runs parallel DA reviews, fixes all issues, checks off the task, and commits. /iterative-review Spawns multiple DA agents in parallel to review a target, then spawns fix agents for all moderate+ issues; loops until no critical, major, or moderate issues remain.

Scripts

scripts-logging Three shell scripts that automatically capture every prompt submitted to Claude Code into per-project Markdown log files, organized by session. scripts-plan Five shell scripts that track plan progress, enforce one-commit-per-task discipline, and audit completed plan runs.
