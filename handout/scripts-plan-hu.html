Összefoglaló

Shell szkriptek a - [ ] / - [x] feladat-checkboxokat tartalmazó Markdown terv-fájlok haladásának követéséhez — közvetlenül a fejlesztők, illetve automatikusan az /implement-next és az /implement-all használja.

Mikor használd

Akkor használd ezeket a scripteket, amikor olyan Markdown terv-fájlokkal dolgozol, amelyek - [ ] / - [x] feladat-checkboxokat tartalmaznak:

  • Ha látni szeretnéd, melyik a következő feladat és mennyi az aktuális készültségi százalék — használd a plan-progress.sh scriptet.

Meghívás

plan-progress.sh

Formázott haladás-fejlécet nyomtat a következő befejezetlen feladathoz. A progress-header-phased.template (fázisokra bontott tervek) vagy a progress-header-flat.template (lapos tervek) sablont használja.

bash ~/.claude/scripts/plan-progress.sh <plan_file>

Kilépési kódok: 0 = fejléc kinyomtatva + NEXT_TASK_* gépileg olvasható sorok; 1 = minden feladat kész; 2 = használati hiba vagy fájl nem található; 3 = nincs felismerhető feladat-szakasz fejléc.

Belső függőségek (a plan-progress.sh hívja — közvetlenül nem indítandók)

  • task_section.awk — AWK töredék, amely akkor aktiválódik, amikor a parser belép egy felismert feladat-szakasz fejlécbe (## Tasks, ## Task vagy ## Task breakdown).
  • progress-header-phased.template — Mustache-stílusú sablon fázisokra bontott tervekhez (ahol vannak ### fázis-fejlécek).
  • progress-header-flat.template — Mustache-stílusú sablon lapos tervekhez (nincsenek fázis-fejlécek).

Függőségi táblázat

Script Típus Belső függőségek Automatikusan hívja
plan-progress.sh user task_section.awk, sablonok /implement-next 1. lépés, /implement-all minden iteráció
task_section.awk internal — plan-progress.sh
progress-header-phased.template internal — plan-progress.sh
progress-header-flat.template internal — plan-progress.sh

1. példa — Terv-előrehaladás ellenőrzése a plan-progress.sh scripttel

Bemenet:

bash ~/.claude/scripts/plan-progress.sh plans/FEAT-001-skill-command-script-handouts.md

Kimenet / Eredmény:

Progress   : [█████████░░░] 79%  (15/19 tasks, 3/6 phases)
Next task  : 4.1  **File**: `handout/scripts-plan.html`

NEXT_TASK_NAME=**File**: `handout/scripts-plan.html`
NEXT_TASK_PHASE=4
NEXT_TASK_NUMBER=1

Kilépési kód: 0. A NEXT_TASK_* sorok gépileg olvashatók, és az /implement-next az 1. lépésben ezeket parsolja, hogy azonosítsa a megvalósítandó feladatot.

Kapcsolódó eszközök

  • /implement-next — Az 1. lépésben meghívja a plan-progress.sh-t.
  • /implement-all — Minden iteráció elején hívja a plan-progress.sh-t.
