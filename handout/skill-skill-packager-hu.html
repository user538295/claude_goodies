/skill-packager

← Vissza az áttekintéshez

Összefoglaló

Egy Claude Code skill mappát feltöltésre kész ZIP fájlba csomagol a Claude Desktop vagy a claude.ai számára, ügyelve arra, hogy a ZIP legfelső szintű mappájának neve megegyezzen a skill name: frontmatter mezőjével (ez az Anthropic feltöltő kötelező követelménye).

Mikor használd

Akkor használd a /skill-packager-t, ha egy Claude Code skillt a Claude Desktopra vagy a claude.aira kell eljuttatnod a Settings → Customize → Skills → Create skill → Upload útvonalon. Ez egyben a kötelező automatikus utólépés is bármely új Claude Code skill létrehozása után — a Desktop ZIP automatikusan elkészül a Code változat mellett.

Kiváltó kifejezések (a SKILL.md-ből):

  • "package this skill"
  • "zip this skill for Desktop"
  • "make the Desktop version"
  • "bundle the skill for upload"
  • "produce the Desktop ZIP"

Automatikusan is kiváltódik utólépésként minden új Claude Code skill létrehozása után, így a Desktop verzió külön kérés nélkül elkészül.

Megjegyzés: a Claude beállításaiban engedélyezni kell a kódfuttatást, különben a Skills szekció rejtett marad a felületen, és a feltöltés meghiúsul.

Meghívás

Slash parancsként hívd meg, opcionálisan megadva a csomagolandó skill mappáját:

/skill-packager [skill-folder-path]

Paraméterek:

  • Skill mappa útvonala — abszolút vagy ~-kezdetű útvonal a skill mappához (pl. ~/.claude/skills/my-skill/). Ha kihagyod, a skill rákérdez, melyiket csomagolja — soha nem találgat.
  • Kimeneti útvonal (opcionális) — ha megadod, oda írja a ZIP-et; egyébként az alapértelmezett ~/.claude/dist/<name>.zip.

A skill ellenőrzi, hogy a mappa tartalmaz-e SKILL.md-t, kiolvassa a name: frontmatter értékét, megerősíti, hogy a mappa neve egyezik vele, majd felépíti a ZIP-et az OS-zaj, build-melléktermékek, titkok és szerkesztői mentések kihagyásával, végül kiírja az útvonalat, méretet, fájlszámot és a három lépéses telepítési útmutatót.

1. példa — Meglévő skill csomagolása (szemléltető példa)

Bemenet:

/skill-packager ~/.claude/skills/aaa/

Kimenet / Eredmény:

Packaging skill: aaa
Source folder : ~/.claude/skills/aaa/
Output ZIP    : ~/.claude/dist/aaa.zip

Building ZIP (excluding __pycache__, .git, .env, .DS_Store, node_modules)...
Done.

ZIP path  : /Users/you/.claude/dist/aaa.zip
Size      : 14 KB
Files     : 2 (SKILL.md, references/)

Install steps:
1. Open Settings → Customize → Skills in Claude Desktop or claude.ai.
2. Click + Create skill → Upload a skill.
3. Select /Users/you/.claude/dist/aaa.zip

2. példa — Név-eltérés elutasítása (szemléltető példa)

Bemenet:

/skill-packager ~/.claude/skills/my-aaa-skill/

(A mappa neve my-aaa-skill, de a SKILL.md-ben name: aaa szerepel.)

Kimenet / Eredmény:

ERROR: Folder name mismatch.
  Folder basename : my-aaa-skill
  SKILL.md name:  : aaa

The Anthropic uploader requires the ZIP's top-level entry to equal the skill's
name: field. Please either:
  (a) Rename the folder to "aaa", or
  (b) Change name: in SKILL.md to "my-aaa-skill".

No ZIP was created.
