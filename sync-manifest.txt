# sync-manifest.txt — the single source of truth for claude_goodies.
#
# These are the "curated" paths: files that live in ~/.claude (runtime) AND
# ship in this repo. Two tools read this one file:
#
#   * claude-sync.sh  — CAPTURE: copies each path  ~/.claude  ->  this checkout
#   * install.sh      — APPLY:   copies each path  this repo  ->  ~/.claude
#
# Format:
#   - One path per line, relative to the repo root (== relative to ~/.claude).
#   - A trailing slash (skills/aaa/) means "directory, copied recursively".
#   - No trailing slash means a single file.
#   - Lines starting with # are comments; blank lines are ignored.
#
# NOT listed here (repo-only, maintained directly in the checkout, never synced
# into ~/.claude): README.md, install.sh, sync-manifest.txt, handout/, assets/.
#
# Junk is never copied regardless of these entries: .DS_Store, __pycache__/,
# *.pyc, .pytest_cache/, node_modules/, .git/, editor swap/backup files.
